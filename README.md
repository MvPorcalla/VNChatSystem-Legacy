### **VNChatSystem-Base**

A modular and reusable **Visual Novel Chat System** built in Unity — designed for messenger-style narrative games.
This base project handles chat UI, message virtualization, object pooling, and data management — making it the foundation for developing interactive story-driven chat experiences.

**Features:**

* Messenger-style chat interface
* Message pooling & virtualization for performance
* Save/load system for dialogue and unlockable content
* Modular managers (ChatManager, VirtualizationManager, PoolingManager, etc.)
* Clean folder structure for scalable game projects

**Purpose:**
Serves as the **core framework** for building VN-style chat simulation games, allowing fast prototyping and consistent system updates across multiple titles.
